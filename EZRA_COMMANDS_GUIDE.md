## 🚀 Quick Start

### Opening Ezra
- **Click the gold orb** (✦) floating in the corner of your screen
- **Drag the orb** to reposition it anywhere on screen
- **Double-click** to quickly open/close

### First Time Using Ezra?
Say: `"Start Quick Quote Wizard"` or click the **Quick Quote** button for a guided walkthrough.

## 🎤 Voice Commands (Platinum+ Feature)

**To activate:** Click the microphone button 🎤 or say `"Hey Ezra"` (if enabled)

### Creating Quotes by Voice

Simply speak naturally. Ezra understands:

**Basic Quote Creation:**
```
"Create a quote for John Smith"
"New quote for borrower Sarah Johnson"
"Build quote: $800,000 property, $400,000 mortgage, needs $100,000 cash"
```

**With All Details:**
```
"Client is Michael Brown, 750 credit score, $600,000 home value, 
 $300,000 mortgage balance, looking for $150,000 HELOC"
```

**Property & Loan Details:**
```
"Property value is $750K"
"Mortgage balance is $350,000"
"They want $100,000 cash out"
"Credit score is 720"
"Investment property"
```

### Voice Shortcuts

| Say This | Result |
|----------|--------|
| `"Go with tier 1"` | Selects highest origination fee tier |
| `"Switch to tier 2"` | Selects middle tier |
| `"Use 15-year fixed"` | Selects 15-year fixed program |
| `"Change to 30-year"` | Switches to 30-year term |
| `"Make it variable"` | Switches to variable rate program |
| `"Highlight that"` | Highlights the selected option |
| `"Undo"` | Reverts last action |
| `"Clear chat"` | Clears conversation history |
| `"Close Ezra"` | Minimizes the widget |

### Rate Input by Voice

```
"For the 4.99 origination fee, the 30-year is 7.45%, 
 20-year is 7.20%, 15-year is 7.20%"

"At 2.99 origination: 30 years at 7.85%, 20 years at 7.60%"

"The 1.50 origination fee: 30-year rate is 8.35%"
```

## ⌨️ Typed Commands & Natural Language

Ezra understands natural language. Type as if you're talking to a colleague.

### Quote Building Commands

| Type This | What Happens |
|-----------|--------------|
| `$800k home, $400k mortgage, need $100k cash` | Auto-fills all borrower fields |
| `Client: John Smith, 750 score, $600K property` | Creates quote with borrower info |
| `Build quote for investment property` | Sets occupancy type |
| `Change cash to $150K` | Updates HELOC amount |
| `Make it 15 years` | Changes term selection |
| `Switch to tier 2` | Changes origination fee tier |

### Deal Structuring Commands

| Type This | What Happens |
|-----------|--------------|
| `Structure this deal` | Analyzes and recommends best program |
| `What's the best program for debt consolidation?` | Recommends based on goal |
| `Calculate CLTV` | Shows combined loan-to-value |
| `What are the payments?` | Calculates monthly payments |
| `Compare fixed vs variable` | Side-by-side comparison |

### Action Commands

| Type This | What Happens |
|-----------|--------------|
| `undo` | Reverts last form change |
| `clear` | Clears chat history |
| `help` | Shows available commands |
| `show shortcuts` | Displays keyboard shortcuts |
| `export quote` | Generates shareable quote link |

### Pasting Lender Data (Platinum+ Feature)

**From Figure Portal:**
1. Go to Figure's Lead Portal
2. Press `Ctrl+A` to select all
3. Copy and paste directly into Ezra
4. Ezra auto-extracts rates, fees, and terms

**Format Ezra understands:**
```
Cash amount
157,156

Origination Fee
1.50%
$2,358

Fixed Rate
$1,210/month
8.35%
30yr fixed
```

## 🤖 AI Model Modes

Click the **AI Mode** button (shows "Fast" by default) to switch between:

| Mode | Color | Best For | Speed |
|------|-------|----------|-------|
| **Fast** | 🔵 Blue | Quick answers, simple questions | ⚡ Instant |
| **Deep** | 🟠 Orange | Calculations, detailed analysis | 🔄 Fast |
| **Complex** | 🟢 Green | Strategy, advanced reasoning | ⏱️ Thorough |

### When to Use Each Mode

**Fast Mode:**
- Quick lookups
- Simple definitions
- Status checks
- "What's the CLTV?"

**Deep Mode:**
- Payment calculations
- Multi-tier comparisons
- Program recommendations
- "Structure this deal"

**Complex Mode:**
- Complex scenarios
- Multiple variables
- Strategic advice
- "Compare all options for this borrower"

## ✨ Special Features

### 1. Deal Radar 📡

**Command:** Click **Lead Briefing** button or type `"Scan for deals"`

Automatically scans your borrower database for:
- Equity opportunities
- Debt consolidation candidates
- Rate reduction possibilities
- Cash-out refinance potential

### 2. Compliance Guardrails ⚠️

**Command:** Click **Compliance Check** button

Paste any SMS or email text to check for:
- TILA/RESPA violations
- Missing disclosures
- Risky language
- Regulatory red flags

### 3. Auto-Fill Magic ✨

Ezra automatically fills quote forms when you:
- Paste lender portal data
- Use voice commands
- Type natural descriptions
- Click tier buttons

**Fields auto-populated:**
- Borrower name
- Credit score
- Property value
- Mortgage balance
- HELOC amount
- All three tier rates
- Origination fees

### 4. Smart Undo

**Command:** Type `"undo"` or `"revert"`

Restores previous form state before:
- Rate imports
- Tier changes
- Auto-fill actions

### 5. Position Memory

Ezra remembers where you place the orb:
- **Bottom Right** (default)
- **Bottom Left**
- **Top Right**
- **Top Left**
- **Custom** (drag anywhere)

## 💡 Pro Tips

### Speed Up Your Workflow

1. **Use natural language** - No need to memorize commands. Just type what you want.

2. **Voice for speed** - Dictate full quotes while reviewing files

3. **Paste for accuracy** - Copy directly from lender portals to eliminate typos

4. **Tier shortcuts** - Click 1️⃣ 2️⃣ 3️⃣ buttons to instantly switch pricing tiers

5. **Undo freely** - Say `"undo"` anytime to revert changes

### Example Workflows

**Quick Quote:**
```
1. Click 🚀 Quick Quote
2. Answer 4 questions
3. Get complete quote
4. Click "Generate Link"
```

**Voice Quote:**
```
1. Click 🎤 microphone
2. Say: "$800K property, $400K mortgage, need $100K cash"
3. Confirm details
4. Say: "Go with tier 2, 15-year fixed"
```

**Portal Import:**
```
1. Open Figure portal
2. Ctrl+A, Ctrl+C
3. Click 📋 Paste Rates button
4. Ctrl+V, click Import
5. All fields auto-filled
```
